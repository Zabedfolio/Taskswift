TaskSwift is a clean, minimal, and premium light-mode task handler designed for high productivity. Built using React and Tailwind CSS v4, it features a polished segmented tab interface, smooth micro-interactions, and real-time progress tracking.

🌐 **Live Demo**: [taskswift.vercel.app](https://taskswift.vercel.app/)

---

## ✨ Features

*   **Premium Light Mode Layout** — Styled with a clean, high-contrast monochrome design inspired by HeroUI (NextUI) and Vercel.
*   **Segmented Control Bar** — Effortlessly filter between **All**, **Active**, and **Completed** tasks with sliding visual states.
*   **Real-time Progress Tracker** — A visual indicator bar displaying task completion percentages.
*   **Tactile Feedback** — Buttons and inputs react dynamically with smooth scaling effects and active hover rings.
*   **Timestamp Badges** — Each task displays a clean creation time badge utilizing Gravity UI clock icons.
*   **One-Click Cleanups** — Easily purge completed tasks with a styled, responsive clear button.
*   **Optimized Performance** — Compiled under Next.js Turbopack for near-instant rendering.

---

## 🛠️ Tech Stack

*   **Core Framework:** Next.js 16 (Turbopack)

## 🚀 Getting Started

### Prerequisites

Ensure you have Node.js (v18 or higher) installed.

### Local Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Zabedfolio/Taskswift.git
   cd taskswift
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the local development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

---

## 📦 Build & Production

To compile a clean optimized production build:

```bash
npm run build
npm run start
```
